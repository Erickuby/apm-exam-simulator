# APM PMQ Exam Simulator

![APM PMQ Exam Simulator Preview](https://github.com/Erickuby/apm-exam-simulator/assets/placeholder.png)

A professional practice assessment tool designed to help candidates prepare for the **APM Project Management Qualification (PMQ)** exam. This simulator provides a realistic exam environment with a full 150-minute timer, 40 high-frequency questions, and detailed performance analytics.

## üöÄ Features

- **Realistic Exam Mode**: Simulates the actual exam conditions with a 2.5-hour timer.
- **Comprehensive Question Bank**: Covers key topics like EVM, Risk, Quality, and Change Control.
- **Instant Feedback**: Detailed explanations for every answer to aid learning.
- **Performance Analytics**: Visual breakdown of results to identify strengths and weaknesses.
- **Modern UI**: Polished, dark-themed interface with glassmorphism effects and smooth animations.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.

## üì¶ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Erickuby/apm-exam-simulator.git
   cd apm-exam-simulator
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## üìù Usage

1. Click **"Start Exam Now"** on the landing page.
2. Answer the 40 questions within the 150-minute time limit.
3. Use the navigation bar to skip or review questions.
4. Submit your exam to see your score and detailed breakdown.
